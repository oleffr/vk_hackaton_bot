# Чат-бот для поддержки первокурсников
## Разработано в рамках хакатона «Цифровой вуз: создай удобный сервис для абитуриентов, студентов и преподавателей» от VK Education и Max

- **Кузьмина Ксения** [KasyKuz](https://github.com/kasykuz)
- **Черникова Василиса** [Ogneva2Vasilisa](https://github.com/Ogneva2Vasilisa)

Этот проект позволяет создавать и обновлять базу знаний для RAG (Retrieval-Augmented Generation) системы на основе веб-страниц и PDF-файлов, а также запускать чат-бота для работы с этой базой.

## Основные возможности решения

- **Создание или обновление базы знаний из PDF-файлов** (`pdfs` директория или указанный путь)
- **Создание или обновление базы знаний из URL** (`seed_urls.txt`)
- **Запуск чат-бота**, который использует базу знаний для ответов на вопросы
- **Справка по командам**, доступная через `main.py`

## Запуск решения через Docker
### Запуск и сборка
Для запуска бота необходимо выполнить только одну команду
```bash
sudo docker-compose up --build
```
**ВНИМАНИЕ! При запуске через Docker никакие другие команды не требуются.**

### Обновление базы знаний бота
При проявлении новых источников данных, база знаний обновляется следующим образом:
1) Останавливаем все контейнеры
2) Удаляем volumes, чтобы пересобрать базу с нуля. Если не требуется, то можно пропустить
```bash
sudo docker-compose down -v
```
3) Запускаем пересборку базы знаний
```bash
sudo docker-compose --profile rebuild up rebuild-kb --build
```

4) Затем снова запускаем основного бота и сервер ИИ
```bash
sudo docker-compose up --build
```
